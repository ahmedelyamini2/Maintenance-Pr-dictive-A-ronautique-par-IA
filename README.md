# Maintenance-Pr-dictive-A-ronautique-par-IA


## Description du projet
Ce projet est un **notebook Google Colab** qui utilise l’**intelligence artificielle** pour la **maintenance prédictive des moteurs d’aéronefs**.  
L’objectif est de **prévoir la durée de vie restante (RUL – Remaining Useful Life)** des moteurs à partir des données des capteurs, afin d’anticiper les défaillances et planifier efficacement la maintenance.

---

## Objectif
- Comprendre et analyser les données des capteurs des moteurs
- Développer un modèle de **Machine Learning** pour prédire la RUL
- Fournir un outil pratique pour la **maintenance prédictive aéronautique**
- Visualiser et comparer les prédictions avec les valeurs réelles pour évaluer la performance du modèle
